# 📋 Lista de Tarefas

Este é um projeto simples de lista de tarefas desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Excluir tarefas
- Salvamento automático no navegador

## 💻 Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 Objetivo
Projeto criado para prática de desenvolvimento web e evolução na área de tecnologia.
